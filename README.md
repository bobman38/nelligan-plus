## How to run the tool locally ?

This is a React frontend application with a NodeJS/Express backend located in ``server/``.


### Launch the backend
```
$ cd server/
$ npm install
$ npm run devstart
```

### Launch the frontend
```
$ npm install
$ npm start
```

And then check on http://localhost:3000
